# snowflake (루트)

## 목적
Claude Code + oh-my-claudecode 기반 워크플로우 템플릿. 이슈 생성부터 PR 머지까지 전 과정을 자동화한다.

## 구조
| 경로 | 역할 |
|------|------|
| `CLAUDE.md` | 아키텍처 불변식·규칙·작업 흐름 (세션 시작 시 필독) |
| `AGENTS.md` | 레포 전체 목차 (지금 이 파일의 상위) |
| `setup.sh` | 템플릿 초기화 스크립트 (1회 실행) |
| `.github/workflows/` | GitHub Actions (Project 보드 자동화) |
| `.claude/` | 커맨드·에이전트·훅 정의 |
| `docs/` | 프로젝트 문서 SOT |
| `scripts/` | 불변식 검증 등 유틸리티 |

## 핵심 워크플로우
```
/bi → /si → /plan → /fi → /ci
```

## 주의
- 커밋·푸시 전 항상 사용자 확인 필요 (CLAUDE.md 행동 규칙)
- 모든 디렉토리에 .ai.md 포함 필수
