# .claude/skills/

## 목적
Claude Code 스킬 저장소. 사용자 프롬프트에서 키워드/의도를 감지해 관련 스킬을 자동 제안한다.

## 구조
```
skills/
├── skill-rules.json              # 스킬 활성화 트리거 규칙 (키워드, 의도 패턴, 우선순위)
├── database-schema-design/       # Snowflake 스키마 설계 스킬
│   └── SKILL.md
├── git-commit/                   # Conventional Commits 자동 생성 스킬
│   └── SKILL.md
├── test-driven-development/      # TDD Red-Green-Refactor 강제 스킬
│   ├── SKILL.md
│   └── testing-anti-patterns.md
└── security-best-practices/      # Snowflake 보안 (RBAC, 마스킹, 네트워크 정책) 스킬
    └── SKILL.md
```

## 작동 방식
1. `UserPromptSubmit` 훅이 사용자 입력을 `skill-activation-prompt.ts`로 전달
2. `skill-rules.json`의 키워드/의도 패턴과 매칭
3. 매칭된 스킬을 우선순위순으로 제안

## 스킬 추가 방법
1. `skills/<skill-name>/SKILL.md` 생성
2. `skill-rules.json`에 트리거 규칙 추가
3. `skills/<skill-name>/.ai.md` 작성
