# security-best-practices/

## 목적
Snowflake 보안 구현 가이드. 웹 앱 범용이 아닌 Snowflake 데이터 플랫폼 전용으로 커스터마이징됨.

## 파일
- `SKILL.md` — RBAC → 네트워크 정책 → 동적 데이터 마스킹 → Row Access Policy → 태깅 → 감사 → Secure Sharing 7단계

## Snowflake 특화 내용
- RBAC 역할 계층 설계, 네트워크 정책 (IP 제한), 마스킹 정책 (PII), Row Access Policy (멀티테넌트), 오브젝트 태깅, ACCESS_HISTORY/LOGIN_HISTORY 감사, Secure View 기반 데이터 공유
