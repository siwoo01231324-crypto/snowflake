# database-schema-design/

## 목적
Snowflake 데이터베이스 스키마 설계 가이드. PostgreSQL/MySQL 범용이 아닌 Snowflake 전용으로 커스터마이징됨.

## 파일
- `SKILL.md` — Snowflake 스키마 설계 5단계 (테이블 정의 → 레이어 아키텍처 → 인제스천 → 클러스터링 → 스트림/태스크)

## Snowflake 특화 내용
- VARIANT/OBJECT/ARRAY 타입, Bronze/Silver/Gold 레이어, Stage/Pipe/COPY INTO, 클러스터링 키, 동적 테이블, Time Travel, Zero-Copy Clone
