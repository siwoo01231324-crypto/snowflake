# .claude/hooks/

## 목적
Claude Code 훅 스크립트 모음. PostToolUse(시크릿 탐지)와 UserPromptSubmit(스킬 활성화) 두 가지 이벤트를 처리한다.

## 파일
- `secret-filter.sh` — 시크릿 패턴 탐지 훅 (API 키, 토큰, 패스워드 등)
- `skill-activation-prompt.sh` — 스킬 활성화 훅 래퍼 (bash → tsx)
- `skill-activation-prompt.ts` — 사용자 프롬프트에서 키워드/의도 패턴을 매칭해 관련 스킬 제안

## 훅 동작

### PostToolUse: secret-filter
- 이벤트: `PostToolUse` (모든 도구 실행 후)
- 탐지 시: `hookSpecificOutput.additionalContext` JSON 출력 + `~/.claude/security.log` 기록
- 미탐지 시: 빈 출력, exit 0
- 항상 exit 0 (PostToolUse는 차단 불가)

### UserPromptSubmit: skill-activation-prompt
- 이벤트: `UserPromptSubmit` (사용자 프롬프트 제출 시)
- stdin으로 `{ "prompt": "..." }` JSON 수신
- `.claude/skills/skill-rules.json`에서 키워드/의도 패턴 매칭
- 매칭된 스킬을 우선순위순으로 출력
- 매칭 없으면 빈 출력, exit 0
- 의존성: `npx tsx` (TypeScript 실행)

## 규칙
- 실제 시크릿 값은 로그에 기록하지 않음
- Python3 미설치 시 secret-filter graceful exit
- 패턴 추가 시 스크립트 내 SECRET_PATTERNS 배열에 추가
- 스킬 트리거 추가 시 `skill-rules.json` 수정
