# .claude/agents/

## 목적
Claude Code 네이티브 서브에이전트 정의 디렉토리.
각 `.md` 파일은 독립적인 에이전트이며, Claude가 `description` 필드를 읽고 자동 호출 시점을 판단한다.

## 구조
| 파일 | 역할 |
|------|------|
| `plan-reviewer.md` | 구현 계획 사전 검토 |
| `code-architecture-reviewer.md` | 코드 품질·아키텍처 리뷰 |
| `documentation-architect.md` | `.ai.md` 및 문서 생성/갱신 |
| `refactor-planner.md` | 리팩토링 분석 및 계획 수립 |
| `code-refactor-master.md` | 리팩토링 실행 및 검증 |
| `frontend-error-fixer.md` | Next.js/React 에러 디버깅 |
| `web-research-specialist.md` | 외부 리서치 전담 |
| `README.md` | 에이전트 카탈로그 및 사용법 |

## 규칙
- 에이전트 파일은 반드시 YAML frontmatter(`name`, `description`) 포함
- `description`은 구체적으로 작성 — Claude가 이걸 읽고 자동 호출 여부를 결정
- 에이전트 추가 시 `README.md`도 함께 갱신
