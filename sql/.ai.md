# sql/ — Snowflake SQL 스크립트

## 목적
Snowflake 데이터베이스 객체(DB, 스키마, 뷰, UDF, Cortex AI)를 정의하는 SQL 스크립트를 관리한다.

## 구조
| 경로 | 역할 | dev_spec 매핑 |
|------|------|--------------|
| `ddl/` | DB·스키마·테이블 생성 DDL | A2-3 |
| `views/` | Marketplace 연동 뷰 생성 (11개) | A2-3 |
| `udf/` | UDF·UDTF 정의 (3개) | B3 |
| `cortex/` | Cortex AI Functions SQL | B4 |
| `validation/` | 데이터 검증 쿼리 | A2-3 #11 |
| `test/` | AC 검증 테스트 쿼리 | 이슈별 TC |

## 번호 규칙
- DDL: `NNN_설명.sql` (001부터 이슈 순서대로)
- 뷰: `NNN_데이터셋_views.sql` (002~006)
- UDF: `함수명.sql`
- 테스트: `test_NN_설명.sql`

## 실행 방법
Snowflake MCP 또는 Snowsight에서 한 문장씩 순서대로 실행한다.
`IF NOT EXISTS` / `CREATE OR REPLACE` 사용으로 멱등성 보장.
