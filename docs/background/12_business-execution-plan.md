# 무빙 인텔리전스 — 사업화 실행 플랜

> 작성일: 2026-04-06
> 목적: 해커톤 프로토타입을 실제 B2B SaaS 사업으로 전환하기 위한 클라이언트 타겟팅, 수익화 전략, 실행 로드맵

---

## Executive Summary

"무빙 인텔리전스"는 통신 주소변경 데이터를 이사의 선행지표로 활용하여, **이사 발생 2~4주 전에 수요를 예측**하고 이를 B2B로 판매하는 데이터 플랫폼이다. 한국 이사 연관 시장(이사+인테리어+가전렌탈+통신전환)은 **연 50조원 이상** 규모이며, 이 시장의 핵심 플레이어들은 "이사 예정 고객을 미리 아는 것"에 높은 지불 의사를 가진다.

**핵심 가설**: 이사 예정 고객 1명의 리드 가치는 최소 5~50만원이다 (업종별 상이). 이 리드를 2~4주 일찍 제공하는 것이 본 사업의 핵심 가치 제안이다.

## 1. 타겟 클라이언트 우선순위

### Tier 1: 즉시 수익화 가능 (0~6개월)

이사와 직접적 매출 연관이 있고, 기존에 이사 고객 획득에 높은 비용을 지출하는 기업군.

#### 1-1. 가전 렌탈사

| 기업 | 매출 | 렌탈 판매량 | 왜 타겟인가 |
|------|------|-----------|-----------|
| **코웨이** | 4.9조원 (2025) | 연 185만 대 | 방문판매(코디) 기반 → 이사 고객 선제 접촉 시 전환율 극대화 |
| **쿠쿠홈시스** | 1.05조원 (2024) | 297만 계정 | 이사 시 정수기·공기청정기 신규 렌탈 수요 급증 |
| **LG전자 렌탈** | 8,600억원 (2023) | — | 렌탈 1조 목표, 이사 고객 확보 경쟁 치열 |
| **SK매직** | Top 3 | — | 코웨이·LG와 점유율 경쟁 중 |

**지불 의사 근거**:
- 코웨이 판관비율 55~60% → 매출 대비 약 2.7~2.9조원이 고객 획득·유지에 사용
- 렌탈 계약 1건당 LTV(생애가치)는 약정 기간(3~5년) × 월 렌탈료(3~5만원) = **108~300만원**
- 이사 고객 1명의 렌탈 전환 가치가 100만원 이상이므로, **리드 1건당 5~10만원** 지불 가능
- 현재 코웨이 코디는 아파트 단지를 무작위 방문 → 이사 예정 단지 정보만 제공해도 효율 2배+

**제안 상품**:
- **API 구독**: 월 300~500만원 — 지역별 이사 예측 데이터 + 세그먼트 정보 피드
- **리드 패키지**: 이사 예정 지역·시기·세그먼트 정보 건당 5~10만원 (CPA 모델)
- **엔터프라이즈 라이선스**: 연 1~3억원 — 전국 데이터 + 커스텀 대시보드 + 전용 API

#### 1-2. 인테리어 플랫폼

| 기업 | 매출 | 왜 타겟인가 |
|------|------|-----------|
| **오늘의집** | 2,879억원 (2024, 첫 흑자) | 시공 거래액 2배 성장 중, 이사 고객 = 인테리어 고객 |
| **집닥** | — | 인테리어 업체 중개, 이사 예정 지역 타겟팅 수요 |
| **하우스** | — | 인테리어 비교 견적 플랫폼 |

**지불 의사 근거**:
- 인테리어 공사 평균 단가: 500~3,000만원
- 오늘의집 수수료율 추정 10~15% → 공사 1건당 50~450만원 수익
- 이사 예정 고객 리드 1건의 가치: **10~50만원** (인테리어 전환율 감안)
- 오늘의집은 CPA 기반 확장 전략 → 이사 예측 데이터 = 정밀 타겟팅 도구

**제안 상품**:
- **타겟팅 데이터 구독**: 월 200~500만원 — 이사 예정 지역 + 아파트 가격대(소득 프록시)
- **CPA 연동**: 이사 예측 데이터 기반 타겟 광고 → 시공 계약 전환 시 건당 10~30만원

#### 1-3. 이사 서비스 플랫폼

| 기업 | 왜 타겟인가 |
|------|-----------|
| **짐싸(Zimssa)** | 이사 O2O 플랫폼, 이사 수요 예측 = 공급 최적화 |
| **숨고** | 생활서비스 매칭, 이사 카테고리 포함 |
| **미소** | 생활서비스 플랫폼, 이사 청소·입주 청소 카테고리 |

**지불 의사 근거**:
- 포장이사 1건당 110~130만원, 플랫폼 수수료 10~20% = 건당 11~26만원 수익
- 수요 예측으로 **차량·인력 사전 배치** → 운영 비용 15~20% 절감 가능
- 이사 성수기(3·9월) 예측 정확도가 곧 수익 최적화

**제안 상품**:
- **수요 예측 API**: 월 100~200만원 — 지역×시기별 이사 수요 지수
- **프리미엄 리드**: 이사 예정 고객 정보 건당 3~5만원

### Tier 2: 중기 수익화 (6~12개월)

데이터 활용 니즈는 높지만, PoC(개념 검증) 과정이 필요한 기업군.

#### 2-1. 통신사 (이탈 방지 + 신규 획득)

| 기업 | 왜 타겟인가 |
|------|-----------|
| **KT** | B2B 빅데이터 서비스 94종 운영, 이사 고객 이탈 방지 수요 |
| **SKT** | 지오비전 운영, 유동인구 데이터 이미 수익화 중 → 시너지 |
| **LG U+** | AI B2B 매출 2028년 2조 목표, 이사 예측 = 자사 고객 유지 도구 |
| **아정당** | 8개 통신사 비교 플랫폼, 이사 = 통신 재가입 핵심 트리거 |

**지불 의사 근거**:
- 통신사 가입자 1인당 ARPU: 월 3~5만원, 약정 2년 기준 LTV: 72~120만원
- 이사 시 통신 해지/전환율 약 30~40% (업계 추정)
- 이사 예정 고객을 미리 파악하면 **이탈 방지 캠페인** 선제 실행 가능
- 경쟁사 고객의 이사 시점 = 최적의 **신규 획득 타이밍**

**제안 상품**:
- **이탈 예측 모듈**: 월 500만~1,000만원 — 자사 고객 중 이사 예정자 식별
- **경쟁사 전환 리드**: 건당 CPA 과금 — 이사 예정 지역 잠재 전환 고객
- **데이터 교환 파트너십**: 아정당 이사 데이터 ↔ 통신사 유동인구 데이터 상호 보완

#### 2-2. 부동산 중개 플랫폼

**제안 상품**:

- **매물 매칭 데이터**: 이사 예정 세그먼트(소득·가구 규모) × 지역 매물 추천

## 3. Go-to-Market 전략

### Phase 1: 해커톤 → PoC 전환 (0~3개월)

**목표**: 해커톤 수상 + 첫 유료 PoC 2~3건 확보

| 액션 | 상세 |
|------|------|
| 해커톤 수상 | 심사위원 중 아정당 CMO(이하석), 리치고 사장(김재구)에게 직접 어필 → 사업 파트너십 논의 기회 확보 |
| 아정당 파트너십 | 이하석 CMO에게 "아정당 데이터의 새로운 수익원"으로 제안 → 데이터 공급 파트너십 체결 |
| PoC 제안서 | 코웨이/쿠쿠 중 1곳 + 오늘의집/집닥 중 1곳에 3개월 무료 PoC 제안 |
| 프로토타입 고도화 | 해커톤 Streamlit 앱 → API 서비스 전환 (Snowflake 기반 유지) |

**해커톤이 사업화의 핵심 레버인 이유**:
- 심사위원 6명 중 4명이 데이터 제공사 관계자 = **잠재 사업 파트너**
- 아정당 CMO(이하석)는 그로스마케팅 전문가 → 이 솔루션의 가치를 즉시 이해할 수 있음
- Snowflake Korea 지사장(최기영) → Snowflake Marketplace 입점 + 공동 마케팅 기회
- 수상 실적 = 초기 고객 설득의 신뢰 자산

### Phase 2: PMF 검증 (3~6개월)

**목표**: PoC → 유료 전환 + 신규 고객 10개 확보

| 액션 | 상세 |
|------|------|
| PoC 결과 분석 | 이사 예측 정확도, 마케팅 전환율 개선 효과를 정량 측정 |
| 가격 검증 | PoC 고객에게 유료 전환 제안, 가격 민감도 테스트 |
| 콜드 아웃리치 | 렌탈사 영업조직 대상 세일즈 (코웨이 지역 지사, 쿠쿠 영업팀) |
| 이사 플랫폼 제휴 | 짐싸, 숨고 등 이사 O2O 플랫폼에 데이터 제공 파트너십 제안 |
| 케이스 스터디 | PoC 결과를 기반으로 "이사 예측 데이터로 마케팅 효율 X% 향상" 케이스 스터디 제작 |

### Phase 3: 스케일업 (6~12개월)

**목표**: 월 매출 1억원 달성 + 시드/프리A 투자 유치

| 액션 | 상세 |
|------|------|
| 통신사 영업 | KT/SKT/LG U+에 이탈 방지 모듈 제안 (Tier 2 공략) |
| 직방 파트너십 | 직방RED와 데이터 교환/번들 제안 (직방의 B2B 성공 경험 활용) |
| Snowflake Marketplace 입점 | "Moving Intelligence" 데이터 상품으로 글로벌 노출 |
| 투자 유치 | ARR 10억원+ 기반으로 시드/프리A 라운드 (목표: 10~20억원) |

### Phase 4: 확장 (12개월+)

| 액션 | 상세 |
|------|------|
| 데이터 소스 확장 | 아정당 외 추가 통신 데이터 소스 확보 (KT/SKT 데이터 직접 계약) |
| 예측 모델 고도화 | 이사뿐 아니라 결혼·출산·퇴직 등 다른 생활 이벤트 예측으로 확장 |
| 해외 진출 | Snowflake Marketplace 통해 동남아 시장 진출 (글로벌 통신 데이터 파트너십) |
| 보험·금융 확장 | 생활 이벤트 기반 보험·대출 상품 타겟팅 데이터 판매 |

## 4. 핵심 리스크 & 대응

| 리스크 | 심각도 | 대응 전략 |
|--------|--------|---------|
| **데이터 확보**: 아정당이 해커톤 후 데이터 제공 중단 | 🔴 Critical | 해커톤 중 이하석 CMO와 사업 파트너십 MOU 논의. 아정당에게 "자사 데이터의 새로운 수익원"임을 증명 |
| **개인정보 규제**: 이사 예측이 개인정보 침해 소지 | 🔴 Critical | 개인 식별 불가한 집계(동 단위) 데이터만 제공. 가명정보 처리 전문 법률 자문 확보. 데이터3법 준수 아키텍처 설계 |
| **예측 정확도**: 이사 예측 모델이 실제로 정확하지 않을 수 있음 | 🟡 High | PoC 단계에서 정확도 검증. "동 단위 이사 수요 증감 트렌드"로 스코프 조정 (개별 이사 예측 → 지역 수요 예측) |
| **대기업 내재화**: 통신사가 직접 이사 데이터 수익화 | 🟡 High | 속도 우위 확보. 통신사는 90%가 효과적 B2B 가격모델 미보유 (EY 보고서). 우리는 분석+서빙 플랫폼에 집중 |
| **시장 교육 비용**: B2B 고객이 이사 예측 데이터 가치를 모름 | 🟡 Medium | PoC 무료 제공 → 성과 증명 → 유료 전환. 케이스 스터디 마케팅 |

## 7. 경쟁 환경 & 차별화

### 직접 경쟁자

한국에서 "이사 예측 데이터"를 B2B로 판매하는 기업은 **현재 없다**.

| 유사 서비스 | 차이점 |
|-----------|--------|
| SKT 지오비전 | 유동인구 데이터(현재 상태). 이사 예측(미래 상태)이 아님 |
| KT 잘나가게 | 상권 분석 도구. 이사 수요와 무관 |
| 직방RED | 부동산 데이터 B2B. 이사 예측 기능 없음 |
| 미국 Porch Group Media | Pre-Mover 데이터 판매 — 미국에서는 이미 검증된 사업 모델. 한국판 없음 |

### 핵심 차별화

1. **선행성**: 이사 "후" 데이터가 아닌 이사 "전" 예측 — 한국 시장 유일
2. **다중 데이터 결합**: 통신(이사 시점) × 부동산(가격대) × 소비(지출 규모) = 풍부한 세그먼트
3. **Snowflake 네이티브**: Marketplace 입점으로 데이터 접근 허들 최소화
4. **검증된 글로벌 모델**: 미국 Pre-Mover 데이터 시장은 이미 수십억 달러 규모 → 한국 로컬라이징
