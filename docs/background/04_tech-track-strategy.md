# 테크트랙 전략 분석

> Snowflake AI & Data Hackathon 2026 Korea — 테크트랙 심사 기준, 심사위원 분석, 파이널 발표 전략

---

## Executive Summary

테크트랙은 개발자/데이터사이언티스트 대상 트랙으로, 기술 혁신성·데이터 활용 능력·문제 해결 능력·비즈니스 임팩트·Snowflake 활용도(가산점) 5개 기준으로 평가된다. 비즈니스트랙 대비 **기술 구현의 깊이와 Snowflake 플랫폼 활용 가산점**이 핵심 차별 요소이며, 심사위원 6명의 전문 분야(클라우드/부동산 데이터/GIS 빅데이터/마케팅 데이터/통신 AI/콘텐츠 AI)를 고려한 맞춤형 어필 전략이 수상 확률을 극대화한다.

## 1. 테크트랙 심사 기준 심층 분석

### 1.1 기술 혁신성 (Technical Innovation)

**정의**: 새로운 기술 활용 및 창의적 구현 방식

**핵심 평가 포인트**:
- 단순 API 호출이 아닌, 독자적인 기술 조합 및 파이프라인 설계
- 최신 AI/ML 기법(LLM, RAG, 에이전트 등)의 실질적 적용
- 기존 접근법 대비 차별화된 아키텍처 또는 알고리즘

**극대화 전략**:
- Snowflake Cortex AI + 외부 모델을 결합한 하이브리드 아키텍처 구성
- 단일 모델 추론이 아닌, 멀티스텝 파이프라인(데이터 수집 → 전처리 → 분석 → 시각화) 구현
- 기술 스택의 각 레이어에서 "왜 이 기술을 선택했는가"에 대한 논리적 설명 준비

### 1.2 데이터 활용 능력 (Data Utilization)

**정의**: 제공 데이터셋의 효과적 활용 수준

**핵심 평가 포인트**:
- 제공 데이터셋(NextTrade/RICHGO/SPH/아정당) 중 다수를 의미 있게 결합
- 데이터 품질 이슈 인지 및 전처리 역량
- 데이터에서 비자명한(non-obvious) 인사이트 도출

**극대화 전략**:
- 최소 2개 이상 데이터셋의 크로스 분석(예: 주식 데이터 + 부동산 데이터 상관관계)
- 데이터 클렌징/변환/피처 엔지니어링 과정을 명시적으로 보여주기
- Snowflake Marketplace 데이터와 제공 데이터셋을 결합하여 부가가치 창출

### 1.3 문제 해결 능력 (Problem-Solving)

**정의**: 명확한 접근 방식과 논리성

**핵심 평가 포인트**:
- 해결하려는 문제의 명확한 정의
- 가설 → 검증 → 결론의 논리적 흐름
- 기술적 제약 인지 및 대안 제시 능력

**극대화 전략**:
- "Problem Statement → Hypothesis → Approach → Result → Impact" 구조로 발표 설계
- 실패한 접근법도 언급하여 문제 해결 과정의 깊이 보여주기
- 정량적 메트릭(정확도, 처리 속도, 비용 절감 등)으로 결과 제시

### 1.4 비즈니스 임팩트 (Business Impact)

**정의**: 실제 가치 창출 가능성

**핵심 평가 포인트**:
- 실제 시장/사용자의 pain point 해결 여부
- 수익 모델 또는 비용 절감 시나리오의 현실성
- 확장 가능성(scalability)과 지속 가능성

**극대화 전략**:
- TAM/SAM/SOM 간략 제시로 시장 규모 어필
- "이 솔루션이 없을 때 vs 있을 때"의 Before/After 시나리오 제시
- 실제 기업/기관이 도입할 수 있는 구체적 시나리오 1~2개 준비

### 1.5 Snowflake 활용도 (가산점)

**정의**: Snowpark, Cortex AI, Snowflake Intelligence, Snowflake Marketplace, Streamlit 등 활용 시 가산점

**핵심 평가 포인트**:
- Snowflake 고유 기능의 실질적(meaningful) 활용
- 단순 데이터 저장이 아닌 연산/분석/서빙 파이프라인에서의 활용
- Snowflake 에코시스템 내 End-to-End 구현

**극대화 전략**:
- **Snowpark**: Python/Java/Scala로 Snowflake 내부에서 데이터 변환 및 ML 모델 학습
- **Cortex AI**: LLM 기반 텍스트 분석, 감성 분석, 요약 등에 Cortex AI 함수 활용
- **Snowflake Intelligence**: 자연어 기반 데이터 질의 인터페이스 구현
- **Snowflake Marketplace**: 외부 데이터셋 연동으로 분석 범위 확장
- **Streamlit**: 인터랙티브 대시보드/데모 앱으로 최종 결과물 시각화

> **핵심**: 가산점 항목은 "사용했다"가 아닌 "핵심 파이프라인에 통합했다"로 어필해야 효과적이다.

## 2. 비즈니스트랙 대비 테크트랙 전략적 장점

### 2.1 트랙별 평가 기준 비교

| 구분 | 테크트랙 | 비즈니스트랙 |
|------|----------|-------------|
| **대상** | 개발자, 데이터사이언티스트 | 비개발자 포함 일반 |
| **제출물** | 소스코드 ZIP + 데모영상(10분) | 프레젠테이션 영상(10분) |
| **평가 1** | 기술 혁신성 | 비즈니스 임팩트 |
| **평가 2** | 데이터 활용 능력 | 데이터 활용 전략 |
| **평가 3** | 문제 해결 능력 | 실행 가능성 |
| **평가 4** | 비즈니스 임팩트 | 독창성 |
| **평가 5** | **Snowflake 활용도 (가산점)** | 프레젠테이션 |
| **상금 1위** | MacBook Pro 14 M5 (24GB/1TB) | MacBook Air 13 M5 (16GB/512GB) |

### 2.2 테크트랙의 전략적 장점

**1. Snowflake 가산점 독점**
- 테크트랙에만 "Snowflake 활용도" 가산점이 존재
- 기술적으로 Snowpark/Cortex AI를 실제 구현하면 경쟁자 대비 확실한 우위 확보
- 비즈니스트랙은 "프레젠테이션" 기준으로 대체되어 가산점 메커니즘 없음

**2. 실물 결과물의 설득력**
- 작동하는 코드와 데모가 "아이디어만 있는" 비즈니스트랙보다 심사위원에게 강한 인상
- 라이브 데모 또는 영상 데모는 "실현 가능성"을 즉각 증명
- 심사위원 Q&A에서 기술적 깊이로 차별화 가능

**3. 경쟁 강도 추정**
- 테크트랙 참가자는 기술 역량 요구로 진입 장벽이 높음
- 비즈니스트랙은 비개발자까지 참가 가능하여 상대적으로 경쟁 풀이 넓을 수 있음
- 그러나 테크트랙 내 경쟁자의 기술 수준은 높으므로, 차별화 포인트가 명확해야 함

**4. 상금 격차**
- 테크트랙 1위: MacBook Pro 14 M5 (24GB/1TB) — 추정 시가 약 319만원
- 비즈니스트랙 1위: MacBook Air 13 M5 (16GB/512GB) — 추정 시가 약 199만원
- 테크트랙 상금이 약 60% 높은 가치

| **주요 경력** | 한국마이크로소프트 COO → 오토데스크코리아 → 어도비코리아 → 구글 클라우드 코리아 사장 → Snowflake Korea 지사장 |

- Snowflake 플랫폼의 **비즈니스 가치**를 강조 — "Snowflake를 쓰면 이런 비즈니스 문제가 해결된다"

- Snowflake 에코시스템(Marketplace, Cortex AI 등) 활용도를 기술적으로 깊이 보여주기
