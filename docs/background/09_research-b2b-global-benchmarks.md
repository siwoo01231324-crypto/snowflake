# 글로벌 B2B 데이터 플랫폼 수익 모델 벤치마크

> 작성일: 2026-04-06
> 목적: 글로벌 B2B 데이터 기업의 실제 수익 모델, 가격 구조, 매출 규모를 분석하여 무빙 인텔리전스(이사/유동인구 데이터) B2B 사업 모델 설계에 시사점을 도출한다.

---

## Executive Summary

글로벌 B2B 데이터 플랫폼 시장은 **구독(Subscription)**, **소비 기반(Consumption-based)**, **라이선스(License)** 세 가지 수익 모델이 주류를 이루고 있다. 조사 대상 7개 기업군의 핵심 패턴은 다음과 같다:

| 기업 | 2025 매출 | 주요 수익 모델 | 핵심 고객 |
|------|-----------|---------------|-----------|
| **Palantir** | $4.5B | 다년 구독+서비스 | 정부기관, 대기업 |
| **Circana (구 IRI+NPD)** | 비공개 (추정 $1.5-2B) | 신디케이트 데이터 구독 | CPG, 리테일러 |
| **Experian** | $7.5B | 구독+트랜잭션 | 금융, 마케팅, 헬스케어 |
| **TransUnion** | $4.6B | 구독+트랜잭션 | 금융, 보험, 마케팅 |
| **Foursquare** | ~$58M | API 구독+데이터 라이선스 | 앱 개발사, 리테일, 광고 |
| **Placer.ai** | 비공개 | SaaS 구독 | 리테일, 부동산(CRE) |
| **Snowflake Marketplace** | $3.6B (전체) | 소비 기반+마켓플레이스 | 데이터 엔지니어링 전반 |
| **Verisk** | $3.1B | 구독 SaaS | 보험, 부동산 |
| **Precisely** | ~$546M | 구독 SaaS | Fortune 100 기업 |
| **Gravy Analytics/Unacast** | 비공개 | API+데이터 피드 구독 | 광고, 리테일, 부동산 |

**핵심 인사이트**: 높은 ACV(연간 계약 금액)를 달성하는 기업일수록 (1) 원천 데이터 + 분석 플랫폼을 결합하고, (2) 산업별 특화 솔루션을 제공하며, (3) 장기 구독 계약(2-5년)을 체결한다.

## 1. Palantir Technologies — 정부/기업 예측 분석 플랫폼

### 수익 모델
- **다년 구독 라이선스**: Gotham(정부), Foundry(기업), AIP(AI 플랫폼) 세 플랫폼을 다년 계약으로 판매
- **가격 구성 요소**: 사용자 수, 데이터 볼륨, 배포 환경(클라우드/온프레미스/에어갭), 전문 서비스 범위에 따라 커스텀 가격 책정
- **대형 계약 중심**: TCV(총 계약 가치) $10M–$100M+ 규모의 다년 계약이 주력

### 매출 규모 (FY2025, 10-K 기반)
- **총 매출**: $4.5B (전년 대비 55% 성장)
- **미국 매출**: Q4 기준 $1.076B (YoY +93%)
  - 미국 정부: $570M (YoY +66%)
  - 미국 상업: $507M (YoY +137%)
- **지역 비중**: 미국 74%, 해외 26%
- **고객 수**: 954개 기업/기관

### 주요 고객 유형
- 정부기관 (미 국방부, 정보기관, 보건 기관)
- 대형 기업 (금융, 헬스케어, 제조, 에너지)

### ACV 및 계약 구조
- **상위 20개 고객 평균 매출**: $93.9M (2024년 $64.6M에서 상승)
- **Q4 2025 신규 TCV**: $4.262B (YoY +138%, 사상 최고)
- **미국 상업 TCV**: $1.34B (YoY +67%)
- **잔여 계약 가치** (2024 말 기준): $5.4B (상업 $3.1B, 정부 $2.3B)

### 핵심 가치 제안
데이터 통합 → 분석 → 의사결정 지원의 풀스택 플랫폼. 특히 AIP를 통한 생성형 AI 통합이 상업 부문 폭발적 성장을 견인.

## 2. Circana (구 IRI + NPD Group) — 소비자 행동 데이터 B2B

### 수익 모델
- **신디케이트 데이터 구독**: POS(판매시점) 데이터를 연간/다년 구독 계약으로 판매. CPG 브랜드와 리테일러에게 12개 이상 채널의 매장 판매 데이터 제공
- **Ad-hoc 데이터 접근**: 구독 외 단건 데이터 구매도 가능
- **분석 솔루션**: Liquid Data Go®(성장형 브랜드용), Price & Promotion(AI 기반 가격 최적화), Complete Why Analytics(판매 성과 분석)
- **컨설팅/자문 서비스**: 데이터 기반 전략 자문

### 매출 규모
- **비상장 기업**: 정확한 매출 미공개
- **추정 규모**: IRI($1.2B, 2021) + NPD($500M+) 합병 기준 $1.5-2B 범위로 추정
- **투자자**: Vestar Capital Partners, Hellman & Friedman, Ares Management 등
- **직원 수**: 약 5,928명

### 주요 고객 유형
- CPG(소비재) 기업: P&G, Unilever 등
- 리테일러: Walmart, Kroger, Target 등
- 식음료, 뷰티, 가전 브랜드

### 가격대
- 대형 CPG 기업: 연간 $500K–$5M+ (데이터 범위, 채널, 카테고리 수에 따라 상이)
- 중소 브랜드: Liquid Data Go®를 통해 낮은 진입점 제공

### 핵심 가치 제안
POS 데이터 + 소비자 패널 데이터 결합으로 "무엇이 팔리는지"뿐 아니라 "왜 팔리는지"까지 분석. 2025년 Nielsen MMM(마케팅 믹스 모델링) 사업부와 NCSolutions 인수로 마케팅 분석 역량 강화.

## 3. Experian — 신용 + 마케팅 데이터 사업

### 수익 모델
Experian은 크게 **B2B**와 **Consumer Services** 두 사업부로 운영하며, B2B 내에서 다음 수익원을 보유:
- **데이터 구독/접근**: 신용 데이터, 소비자 프로파일, 대안 데이터 구독
- **Decision Analytics**: 의사결정 지원 소프트웨어 SaaS (사기 탐지, 신원 확인, 신용 평가)
- **Marketing Services**: 소비자 타겟팅, 광고 기술, ID 그래프 기반 마케팅 솔루션
- **트랜잭션 기반**: 신용 조회 건당 과금

### 매출 규모 (FY2025, 2025년 3월 결산)
- **총 매출**: $7,507M (전년 대비 +8%, 유기적 성장 +7%)
- **B2B**: 유기적 성장 +6% (분석, 모기지, 대안 데이터, 수직 시장 성장)
- **Consumer Services**: 유기적 성장 +7% (2억+ 무료 회원 기반)
- **수직 시장(Verticals)**: 전체 매출의 20% 이상 (Marketing Services, Automotive, Health 포함)

### 5대 생태계
1. **Financial Services**: 신용 평가, 대출 심사
2. **Consumer Services**: 신용 모니터링, 신원 보호
3. **Health**: 의료 청구, 환자 확인
4. **Automotive**: 차량 이력, 딜러 솔루션
5. **Marketing Services**: 디지털 광고 타겟팅, 커넥티드 TV, ID 그래프

### 주요 고객 유형
- 금융 기관 (은행, 보험사, 핀테크)
- 광고주, 에이전시, 퍼블리셔
- 의료 기관, 자동차 딜러

### 핵심 가치 제안
세계 최대 소비자 데이터 자산을 바탕으로 신용 데이터를 넘어 마케팅, 헬스케어, 자동차 등 다양한 수직 시장으로 B2B 데이터 사업 확장.

## 6. Snowflake Marketplace — 데이터 공유/판매 수익 구조

### 수익 모델 (플랫폼 사업자 관점)
Snowflake 자체는 **소비 기반 과금(consumption-based pricing)** 모델:
- **컴퓨팅**: 쿼리 실행 시 크레딧 소비
- **스토리지**: 저장 데이터량에 따라 과금
- **데이터 전송**: 리전 간 데이터 이동 시 과금

### Marketplace 수익 구조 (데이터 제공자 관점)
Snowflake Marketplace는 데이터 제공자(Provider)가 세 가지 가격 모델 중 선택:

1. **사용량 기반(Usage-Based)**:
   - 쿼리당 고정 요금
   - 월별 접근 시 월 고정 요금
   - 커스텀 이벤트 기반 과금 (앱 전용)
   - 최대 월 요금 상한 설정 필수

2. **구독 기반(Subscription)**:
   - 반복 구독 (1–36개월, 자동 갱신)
   - 비반복 구독 (일회성 결제)

3. **할부(Installment)**:
   - 구독 비용을 분할 결제

### 결제 처리
- Stripe를 통해 소비자 결제 → 제공자에게 직접 지급
- 소비자가 Snowflake 용량 커밋으로 구매 시 Snowflake가 제공자에게 지급

### 매출 규모 (FY2025, 2025년 1월 결산)
- **총 매출**: $3.63B (YoY +29%)
- **제품 매출**: $3.46B
- **순 매출 유지율(NRR)**: 126%
- **잔여 이행 의무(RPO)**: $6.9B (YoY +33%)
- **Marketplace 규모**: 820+ 제공자, 3,400+ 데이터/앱 목록 (2025 중반 기준)
- **데이터 공유 고객 비중**: 전체 고객의 약 1/3이 데이터 공유 활용

### 핵심 가치 제안
데이터 복사/이동 없이 실시간 데이터 공유 → 네트워크 효과. 제공자는 자체 빌링 시스템 불필요. 소비자는 기존 Snowflake 환경에서 바로 외부 데이터 접근.

## 7. Verisk / Precisely — 보험·부동산 데이터 분석 B2B SaaS

### 7-1. Verisk Analytics

#### 수익 모델
- **구독 SaaS**: 보험 언더라이팅, 클레임 관리, 재해 리스크 모델링 소프트웨어
- **데이터 라이선스**: 보험 요율, 손실 비용, 양식/규정 데이터
- **트랜잭션 기반**: 건별 조회/평가 과금 (재산 견적, 사기 탐지 등)

#### 매출 규모 (FY2025)
- **총 매출**: $3,073M (전년 대비 +6.6%)
- **언더라이팅 매출**: $2,180M (YoY +7.7%)
  - 양식/규정/손실비용 서비스, 재해/리스크 솔루션, 생명보험, 특수보험 솔루션
- **클레임 매출**: $893M (YoY +4.1%)
  - 사기 방지, 재산 견적, 상해 솔루션
- **조정 EBITDA**: Q1 $417M (YoY +9.5%)

#### 전략적 인수
- **SuranceBay**: $163M (2025년 7월) — 보험 유통 플랫폼
- **AccuLynx**: $2.35B (계약 체결) — 루핑/건설 업계 소프트웨어

#### 주요 고객
- 보험사 (P&C, 생명, 건강)
- 재보험사
- 부동산/건설 업계
- 정부 규제 기관

#### 핵심 가치 제안
미국 보험 산업의 사실상 표준(de facto standard) 데이터 인프라. 보험 요율, 클레임 데이터, 재해 모델링에서 독점적 지위.

### 7-2. Precisely

#### 수익 모델
- **SaaS 구독**: 데이터 무결성(Data Integrity) 소프트웨어
- **주요 제품군**: 데이터 통합, 데이터 품질, 데이터 거버넌스, 위치 인텔리전스, 데이터 강화(enrichment)
- **구독 + 서비스**: 연간 구독 + 구현/컨설팅 서비스

#### 매출 규모
- **추정 매출**: ~$546M
- **고객 수**: 12,000+ 기업, 100+ 국가
- **Fortune 100 중 고객**: 90% (95개사)
- **소유 구조**: Clearlake Capital + TA Associates (다수 지분), Centerbridge Partners (소수 지분)

#### 핵심 가치 제안
데이터 정확성·일관성·맥락 제공 — AI/자동화 시대에 "데이터 무결성"의 핵심 인프라. 위치 인텔리전스 제품을 통해 지리 공간 데이터 강화.

위치 데이터 B2B 시장에서 매출 과장의 위험성과 데이터 프라이버시 리스크를 보여주는 사례.

### Gravy Analytics + Unacast (합병)

## 무빙 인텔리전스(이사/유동인구 데이터) 적용 시사점

### 1. 수익 모델 설계

**추천: 하이브리드 모델 (구독 기반 + 소비량 연동)**
- **기본 구독**: 대시보드 접근, 기본 분석 리포트 (월/연간 구독)
- **API 과금**: 쿼리/호출 건당 추가 과금 (Snowflake Marketplace 모델 참고)
- **프리미엄 분석**: 예측 모델, 커스텀 리포트 등 고부가가치 분석은 상위 티어

### 2. 가격 전략

**Placer.ai의 PLG 전략 + Verisk의 산업 표준 전략 결합**
- Free tier로 시장 인지도 확보 (Placer.ai 모델)
- 산업별 특화 솔루션으로 높은 ACV 확보 (부동산, 리테일, 물류)
- 데이터 제공 범위(지역, 기간, 해상도)에 따른 가격 차등

### 3. 고객 세그먼트

| 세그먼트 | 예상 ACV | 핵심 가치 |
|----------|----------|-----------|
| 부동산 개발/투자사 | $100K–$500K | 상권 분석, 인구 유입/유출 예측 |
| 리테일 체인 | $50K–$200K | 매장 입지, 유동인구 변화 |
| 이사/물류 기업 | $30K–$100K | 이사 수요 예측, 경로 최적화 |
| 지방 정부/공공기관 | $50K–$300K | 인구 이동 패턴, 도시 계획 |
| 금융기관 (은행/보험) | $100K–$500K | 지점 입지, 리스크 평가 |
| 광고/마케팅 기업 | $20K–$100K | 타겟팅, 리타겟팅 |

### 4. 핵심 교훈

1. **데이터 독점성이 ACV를 결정한다**: Verisk(보험), Circana(CPG)처럼 해당 산업에서 "표준 데이터"가 되면 높은 가격 책정 가능
2. **플랫폼 + 데이터 결합이 핵심**: 원천 데이터만으로는 차별화 어려움. Palantir처럼 분석 플랫폼과 결합해야 높은 가치
3. **네트워크 효과를 설계하라**: Snowflake Marketplace처럼 데이터 제공자-소비자 양면 시장 구축 시 기하급수적 성장 가능
4. **프라이버시 리스크 관리 필수**: Near Intelligence의 파산, Gravy Analytics의 데이터 침해 사례에서 보듯 위치/이동 데이터의 프라이버시 리스크는 사업 존폐를 좌우
5. **장기 계약으로 ARR 안정화**: Palantir, Verisk 모두 다년 계약으로 매출 예측 가능성 확보
