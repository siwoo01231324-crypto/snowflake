# B2B 데이터 SaaS 가격 벤치마크 및 유닛 이코노믹스

## 1. B2B 데이터 SaaS 가격 사례 (2024–2026)

### 1-1. Intent / 위치 데이터 플랫폼

| 업체 | 카테고리 | 가격 구조 | 연간 비용 범위 |
|------|----------|-----------|---------------|
| **ZoomInfo** | B2B 연락처 + Intent | 연간 계약 전용, 좌석 수·기능·크레딧 기반 커스텀 견적 | $15,000–$30,000 (팀) / $30,000–$60,000+ (Enterprise 25석+) |
| **Bombora** | B2B Intent Data | 연간 계약 전용, 1년 이상. 기본 Company Surge 플랜 | $25,000–$30,000/yr 시작, 대부분 $50,000–$100,000/yr |
| **6sense** | ABM + Revenue AI | 연간·2년 계약. 모듈(Sales Intelligence + Predictive AI + Credits) 조합 | $50,000–$300,000/yr. 중간값(median buyer) $55,211/yr |
| **Placer.ai** | 위치 인텔리전스 / 유동인구 | 12개월 약정. 팀 규모·산업·기능 기반 커스텀 | Starter: $500/월(연납), Enterprise: $수천/월 |
| **Similarweb** | 디지털 인텔리전스 | 티어 기반 커스텀. 셀프서비스 $199/월부터 | $15,000–$75,000/yr (일반), Enterprise: $75,000–$200,000+/yr |
| **Clearbit (→ Breeze Intelligence)** | 데이터 인리치먼트 | 2023년 HubSpot에 인수. 2025년 4월 standalone API 중단. HubSpot 내 크레딧 기반 | HubSpot Starter + 100크레딧: $75/월 시작; 미드마켓: $1,000–$5,000+/월 |

**핵심 관찰:**
- B2B 데이터 플랫폼은 거의 예외 없이 **연간 계약 전용(no monthly option)**
- 대부분 가격 비공개 + 커스텀 견적. 협상에서 평균 12% 할인 가능(6sense 기준)
- Enterprise 딜: 구현 비용 $5,000–$50,000 별도 청구 사례 존재

### 1-2. Pre-Mover / 이사 데이터 전문 업체 (미국 시장)

| 업체 | 제품 | 가격 구조 | 구체적 수치 |
|------|------|-----------|------------|
| **Alesco Data** | New Mover + Pre-Mover DB (월 70만 건) | 건당(per record) + 연간 전체 DB 라이선스 | $0.02/record (=**$20/천 건**) ~ $250,000/yr (전체 DB 라이선스) |
| **Speedeon Data** | Mover + Life Event Data (2억+ 개인) | 1회성 캠페인 구매 또는 연간/월간 라이선스 + 사용량 기반 | 공개 가격 없음. 커스텀 견적. 연간 라이선스 또는 일회성 구매 |
| **Deluxe** | Pre-Mover + New Mover (단일 소스 대비 4배 커버리지) | 커스텀 견적 전용. 연간 계약 | 공개 가격 없음. 대형 은행·통신사·리테일 체인 대상 엔터프라이즈 딜 |
| **Porch Group Media** | USPS + 독자 소스 기반 pre/new mover | 커스텀 | 공개 가격 없음 |

**이사 리드 CPL(Cost Per Lead) 맥락:**
- B2B 전반 평균 CPL: $84(전체 채널), Google Ads $70, LinkedIn $110 (2025 기준)
- 부동산·금융서비스 등 고가치 산업: CPL $400+
- Mover 리드는 "life event trigger" 특성상 일반 CPL 대비 프리미엄 적용됨 (구체적 공개 벤치마크 없음, 업체별 견적 필요)

## 2. B2B SaaS 유닛 이코노믹스 벤치마크 (2025)

### 2-1. CAC (Customer Acquisition Cost) by Segment

| 세그먼트 | CAC 범위 | 비고 |
|----------|----------|------|
| **SMB** | $200–$300 | 단순 셀프서브, 짧은 사이클 |
| **Mid-Market** | $1,407–$5,330 | 3–4개월 영업 사이클, PoC 포함 |
| **Enterprise** | $2,206–$14,774 | 6–9개월 사이클, 복수 의사결정자 |
| 전체 평균 (모든 채널) | ~$1,200 | 2019 대비 222% 상승 |
| Enterprise (Fintech) | ~$14,774 | 최고 수준 |
| Enterprise (eCommerce) | ~$2,206 | 최저 수준 |

**채널별 CAC (2025):**
- Referral: $150 (가장 효율적)
- Organic search: $480–$942
- Paid search (Google): $802
- Paid social (LinkedIn): $110+ (CPL 기준)

**CAC Ratio (신규 ARR $1 획득에 드는 S&M 비용):**
- 중간값: $2.00 (2024년, 전년 대비 14% 상승)
- 하위 4분위: $2.82
- 상위 1분위: $1.00

### 2-2. CAC Payback Period

| 기준 | 수치 |
|------|------|
| 중간값 (2024) | **20개월** |
| 건강한 목표 | 12–15개월 이하 |
| 역사적 평균 (2019 이전) | 12–14개월 |
| 공개 클라우드 기업 평균 | 39개월 |
| 권고: 위험 신호 | 16개월+ |

### 2-3. LTV:CAC Ratio

| 기준 | 수치 |
|------|------|
| 2024 중간값 | **3.6:1** |
| 지속 가능 최소치 | 3:1 |
| 즉각 문제 신호 | 2:1 이하 |

### 2-4. NRR (Net Revenue Retention) by Segment

| 세그먼트 | NRR 범위 | 비고 |
|----------|----------|------|
| **SMB** | 97% | 이탈률 높음 |
| **Mid-Market** | 108% | |
| **Enterprise** | 118% | |
| 상위 1분위 | >130% | |
| 전체 중간값 (2025) | 106% | 2024 대비 소폭 개선 |
| 건강한 기준 | 100%+ | expansion > churn |

**NRR의 성장 레버리지:**
- NRR 100% 미만 → 100–110% 개선 시 성장률 +5%p
- ARR $50M+ 기업: 전체 신규 ARR의 50%+ 가 기존 고객 expansion에서 발생

### 2-5. Gross Margin

| 구분 | 수치 |
|------|------|
| 소프트웨어 총마진 (중간값, $1M–$5M ARR) | **77%** |
| 소프트웨어 총마진 (대형, $50M+ ARR) | ~80% |
| 서비스 포함 총마진 | 71–72% |
| 위험 신호 | 70% 미만 |

**데이터/분석 SaaS 특성:** 순수 소프트웨어 마진(75–85%)보다 데이터 수집·갱신 비용으로 인해 5–10%p 낮을 수 있음.

### 4-2. 한국 엔터프라이즈 조달 특성 (글로벌 벤치마크 기반 추정)

직접적인 한국 전용 B2B SaaS 계약 규모·영업 사이클 공개 데이터는 미확인. 아래는 글로벌 벤치마크 + 한국 시장 관찰 종합:

| 항목 | 글로벌 엔터프라이즈 기준 | 한국 시장 추정 |
|------|----------------------|--------------|
| 엔터프라이즈 영업 사이클 | 6–9개월 ($100K+ 딜) | 6–12개월 (조달 절차·내부 결재 추가) |
| 미드마켓 영업 사이클 | 3–4개월 | 4–6개월 |
| 평균 ACV (엔터프라이즈) | $100,000+ | 대기업: 1억–3억 원 / 중견기업: 3,000만–1억 원 |
| PoC → 유료 전환율 (엔터프라이즈) | 20–30% (일반 SaaS free trial 기준) | 데이터 딜: 15–25% (복잡한 기술 검증 포함) |
| 구매 의사결정자 수 | 대기업 6–10명 | 한국 대기업: IT + 구매 + 현업 + 법무 등 7–12명 |

**한국 특유 조달 리스크:**
- 공공기관: 조달청(나라장터) 등록 필요. 단가 계약 기준 마련에 6–12개월 소요 가능
- 대기업: 보안 심의·망분리 요건 강함 → PoC 기간 길어짐
- 중견기업: 예산 사이클(연초 Q1, 하반기 Q3)에 집중 → 영업 타이밍 중요
