# Snowflake AI & Data Hackathon 2026 Korea — 공식 정보

## 개요

**Snowflake AI & Data Hackathon 2026**은 데이터와 AI를 활용해 혁신적인 솔루션을 만드는 해커톤이다.
개발자부터 비즈니스 아이디어를 가진 비개발자까지 모두 참여 가능하다.

> "여러분의 아이디어가 가치가 되는 순간, 데이터와 AI로 만드는 나만의 앱"

## 트랙

### 테크(개발) 트랙
- 대상: 데이터 사이언티스트, 엔지니어, 개발자
- 지원 언어: Python, Java, Scala
- 제출물: 소스 코드(ZIP) + 데모 영상(10분 이내)

### 비즈니스(아이디어) 트랙
- 대상: 비즈니스 아이디어를 가진 누구나 (개발 경험 불필요)
- 제출물: 프레젠테이션 영상(10분 이내)

---

## 평가 기준

### 테크 트랙
1. 기술 혁신성 — 새로운 기술 활용, 창의적 구현
2. 데이터 활용 능력 — 제공 데이터 효과적 활용
3. 문제 해결 능력 — 명확한 접근 방식, 논리성
4. 비즈니스 임팩트 — 실제 가치 창출 가능성
5. Snowflake 활용도 — Snowpark, Cortex AI 등 활용 시 가산점

### 비즈니스 트랙
1. 비즈니스 임팩트 — 시장 가치, 실용성
2. 데이터 활용 전략 — 제공 데이터 기반 의사결정
3. 실행 가능성 — 실현 가능한 로드맵
4. 독창성 — 새로운 관점, 창의적 아이디어
5. 프레젠테이션 — 명확한 커뮤니케이션

## 데이터셋

모든 데이터는 [Snowflake Marketplace](https://app.snowflake.com/marketplace)에서 확인 가능.

### 1. NextTrade — 주식 시장 데이터
- 우선호가 정보 (10단계 매도/매수), 종가 매매 호가
- 체결 정보 (가격, 거래량, 거래대금), 등락률, 프로그램매매 통계
- 종목 기본 정보

### 2. RICHGO — 부동산 데이터
- 아파트 단지 기본 정보 (서울시 100세대 이상)
- AI 시세 (단지별·지역별) — 2021-01 ~ 2026-02
- 지하철역사 거리 및 승하차수
- 인구 이동자수 — 2011-01 ~ 2026-01
- 시도 인구 전입전출 — 2012-01 ~ 2026-01

### 3. SPH — 지역 빅데이터
- SKT 유동인구 (서초구·영등포구·중구) — 2021~2025년
- KCB 자산/소득 (동일 지역) — 월별 현황
- 신한카드 소비 내역 (동일 지역) — 월별 트렌드
- 행정동경계 정보 (법정동 기준)

### 4. 아정당 — 통신 데이터
- 계약 트렌드 — 2024-01 ~ 현재
- 이사/이동 추정 데이터, 렌탈 가전 데이터
- 고객 세그먼트 및 마케팅 통계
- CS 운영 통계 — 2022-08 ~ 현재

## 제출 요건

### 테크 트랙
1. 과제 제출 양식 (제공 폼 작성)
2. 과제 템플릿 (파워포인트)
3. 소스 코드 — ZIP 파일 다운로드 링크
4. 데모 영상 — 10분 이내 (코드 설명, 예측 모델, 앱 사용 방법, 솔루션·인사이트, 향후 전략 포함)

### 비즈니스 트랙
1. 과제 제출 양식 (제공 폼 작성)
2. 과제 템플릿 (파워포인트)
3. 프레젠테이션 영상 — 10분 이내 (솔루션·인사이트, 향후 전략 포함)

### 결선 진출 시 추가
- 발표 10분 + 질의응답 3분
- 마지막 제출물만 심사 (수정 제출 가능)

## 참가 자격 및 규칙

- 한국 거주자, 나이 제한 없음, 사전 기술 경험 불필요
- 팀 구성: 1인 이상 4인 이하, 모든 팀원 개별 신청 필수 (동일 팀명으로 등록)
- 팀 변경은 신청 기간(~2026-04-05) 내에만 가능, 이후 불가
- Snowflake 플랫폼 반드시 포함 (최소 기본 활용)
- 추가 오픈소스·API·SDK 자유 사용 가능
- 개발 결과물 참가자 소유 (Snowflake 우수 사례 소개 시 별도 동의)
- 참가비 무료
