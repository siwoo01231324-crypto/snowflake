# 한국 시장 B2B 데이터 사업 사례 및 이사 관련 시장 분석

## Executive Summary

한국은 통신 3사(KT, SKT, LG U+)가 빅데이터 수익화를 적극 추진하며, 2024년 합산 매출 59조 원 중 비통신/B2B 사업 비중을 50%까지 확대하려는 전략을 실행 중이다. 프롭테크 영역에서는 직방이 B2B 데이터 솔루션 '직방RED'로 2025년 매출 116억 원(전년 대비 970% 성장)을 달성하며, 부동산 데이터의 B2B 수익화 가능성을 입증했다.

이사 관련 시장은 **연간 628만 명이 이동**하며, 이사업체 시장만 **최소 3조~3.5조 원**, 인테리어/리모델링 시장 **37조 원**, 가전렌탈 시장 **12조 원 이상**으로 이사를 트리거로 한 연관 소비 시장은 **50조 원 이상** 규모다. 이사 고객은 통신 재가입, 가전 구매/렌탈, 인테리어 시공 등 다수의 고가 의사결정을 단기간에 집중적으로 수행하므로, "이사 인텔리전스" 데이터의 B2B 가치는 매우 높다.

## 2. 한국 부동산 테크(PropTech) B2B 사업

### 2.1 직방: B2B 데이터 수익화의 성공 사례

| 지표 | 수치 |
|---|---|
| 2024년 매출 | 12억 원 |
| 2025년 매출 | **116억 원** (전년 대비 **970% 성장**) |
| 2025년 월 매출 추이 | 3억 원 -> 15억 원 |
| 2026년 매출 목표 | 전년 대비 76% 성장, 4분기 월 매출 73억 원 |

**직방RED (Real Estate Data):**
- 부동산 전문가를 위한 AI 빅데이터 솔루션
- 태블로(Tableau) 임베디드 대시보드 기반 시각화 리포트
- 공공 데이터 + 직방 고객 거래 데이터 통합 분석
- 분양 경쟁률 예측, 상권 분석, 가격 추이 등 제공
- **주요 고객**: 건설사(미분양 타겟 광고), 자산운용사, 은행 등 금융사
- 미분양 물건을 직방 가입자 데이터 기반 타겟팅하는 광고 상품이 급성장 동력

**호갱노노** (2018년 직방 인수):
- 아파트/오피스텔 부동산 정보 플랫폼 1위
- 국토교통부 실거래가 데이터 기반 가격 변화 제공

### 2.2 부동산114 (HDC랩스 인수)

| 지표 | 수치 |
|---|---|
| 연간 매출 | 112억 원 (핵심 사업 수익 92억 원) |
| 데이터 축적 기간 | 25년 이상 |
| 인수 주체 | HDC랩스 (2025년 영업양수) |
| 양수도 대금 | 6억 3,600만 원 |

**B2B 데이터 솔루션:**
- **REPS**: 주거용 통계분석 시스템 (건설, 금융, 공기관, 회계, 감정평가 등에서 활용)
- **RCS**: 상업용 통계분석 시스템 (2021년~)
- **스마트R**: 중개 솔루션
- 전국망 중개사 네트워크 활용 B2C/B2B 채널 확대 계획

### 2.3 KB부동산 데이터허브

- KB가 운영하는 부동산 데이터 플랫폼
- KB통계(자체 조사/집계) + 공공통계(통계청, 국토부, 한국부동산원) 통합 제공
- 금융 빅데이터 플랫폼과 연계한 데이터 라이선스 모델

### 2.4 프롭테크 시장 특성

- 한국 프롭테크 기업의 **65%가 부동산 마케팅 플랫폼** 형태
- 데이터3법 이후 빅데이터 협업 기반 B2B 수익화 가속
- 공인중개사 광고 수익 의존에서 **데이터 기반 B2B SaaS**로 전환 중

## 4. 이사 고객 획득 비용 (CAC) 분석

### 4.1 코웨이 마케팅 비용 구조

| 지표 | 수치 |
|---|---|
| 2024년 연간 매출 | 4조 3,101억 원 |
| 2024년 영업이익 | 7,954억 원 (영업이익률 약 18.5%) |
| 2025년 연간 매출 | 4조 9,636억 원 |
| 2025년 영업이익 | 8,787억 원 (영업이익률 약 17.7%) |
| 2025년 렌탈 판매량 | 185만 대 |

코웨이는 전통적으로 **방문판매(코디) 인력** 기반의 고객 획득 구조를 유지하고 있으며, 판매관리비 중 인건비(방문판매 코디 수당)와 광고선전비가 주요 CAC 항목이다. 영업이익률 약 18%를 감안할 때, 매출의 상당 부분이 고객 획득 및 유지 비용으로 지출됨을 추정할 수 있다.

**추정 CAC (업계 분석 기반):**
- 코웨이의 판관비율은 매출 대비 약 55~60% 수준으로 알려져 있음
- 연간 렌탈 판매량 185만 대 기준, 마케팅/영업 비용의 상당 부분이 **이사 고객 타겟팅**에 집중
- 방문판매 코디 1인당 관리 계정 수 약 200~300개

### 4.2 인테리어 플랫폼 CAC

**오늘의집:**
- CPA(Cost Per Action) 중심 확장 전략 -> 볼륨 확대 한계에 빠름
- 커머스 수수료 + 광고비가 수익 모델
- 2024년 매출 2,879억 원 중 광고 매출 비중 확대가 흑자 전환의 핵심
- 시공 중개 거래액 2배 증가로 새로운 수익원 확보

**집닥:**
- 인테리어 업체 중개 플랫폼
- 오늘의집과 사업 영역 겹침이 심화되며 경쟁 치열

### 4.3 이사업체의 마케팅 비용

- 이사 시장은 대부분 **3~5인 규모의 영세 자영업체**로 구성
- 온라인 플랫폼(숨고, 미소 등)을 통한 고객 유치 비용이 증가 추세
- 포장이사 업체의 주요 마케팅 채널: 네이버 키워드 광고, 이사 비교 플랫폼

### 4.4 이사 고객의 B2B 가치

이사 고객 1명이 트리거하는 연관 소비를 합산하면:

| 항목 | 평균 지출 | 비고 |
|---|---|---|
| 포장이사 비용 | 100~130만 원 | 24~34평 기준 |
| 인테리어/리모델링 | 500~3,000만 원 | 부분~전체 리모델링 |
| 가전 구매/렌탈 | 200~500만 원 | 정수기, 에어컨, 세탁기 등 |
| 통신/인터넷/IPTV 재가입 | 약정 기준 120~300만 원 | 2~3년 약정 총액 |
| 가구/생활용품 | 100~500만 원 | 이사 후 구매 |
| **1건당 총 연관 소비** | **약 1,000만~4,000만 원** | 규모에 따라 차이 |

> 연간 628만 이동자 기준, 세대 수로 환산하면 약 200~250만 세대가 이사하며, **이사 1건당 연관 소비가 최소 1,000만 원 이상**임을 감안하면 이사 트리거 연관 소비 시장은 **20조~100조 원** 규모로 추산된다.

- 국내외 가입 고객사: **10만 곳 이상** (한국, 일본, 미국)

**마켓보로:**

## 6. 이사 관련 시장 총정리

### 6.1 시장 규모 종합

| 시장 세그먼트 | 규모 | 성장 전망 |
|---|---|---|
| 이사업체(포장이사) 시장 | 3조~3.5조 원 (공식) / 20조 원 (업계) | 이사 건수 연 2~3% 증가 추세 |
| 인테리어/리모델링 시장 | 37~49조 원 | 2030년 46조 원 (유지보수만) |
| 가전렌탈 시장 | 12조 원 | 2025년 전체 렌탈 100조 원 |
| 통신/인터넷/IPTV | 약 4~5조 원 (이사 연계 재가입 추정) | IPTV 성장 둔화, AI 서비스 성장 |
| 가구/생활용품 | 수조 원 규모 | 홈퍼니싱 트렌드 확대 |
| **이사 트리거 연관 소비 총합** | **최소 50조 원 이상** | 지속 성장 |

### 6.2 이사 고객의 특성

1. **높은 구매력**: 이사 1건당 최소 1,000만 원 이상 연관 소비
2. **집중된 의사결정 시점**: 이사 전후 1~3개월에 다수의 구매 결정이 집중
3. **예측 가능성**: 전입신고 데이터, 부동산 거래 데이터 등으로 이사 시점 예측 가능
4. **반복 발생**: 20~30대 이동률 21~24%로, 젊은 세대일수록 이사 빈도 높음
5. **다채널 소비**: 오프라인(이사업체, 가전매장) + 온라인(오늘의집, 렌탈) 동시 활용

## 7. 무빙 인텔리전스(Moving Intelligence) 한국 시장 기회

### 7.1 데이터 소스의 풍부함

한국은 다음과 같은 이사 관련 데이터가 체계적으로 축적되어 있어, "무빙 인텔리전스" 데이터 상품의 구축에 유리하다:

- **통계청 국내인구이동통계**: 전입신고 기반, 읍면동 단위 이동 데이터
- **국토교통부 실거래가 시스템**: 부동산 매매/전월세 거래 실시간 공개
- **통신사 유동인구 데이터**: SKT 지오비전 등 2,700만 가입자 기반
- **부동산 플랫폼 데이터**: 직방, 호갱노노, 부동산114 등의 검색/관심/거래 데이터
- **공공데이터 포털**: 건축 인허가, 아파트 관리비, 에너지 사용량 등

### 7.2 잠재 B2B 고객과 수익 기회

| B2B 고객 | 니즈 | 예상 지불 의향 |
|---|---|---|
| **가전렌탈 기업** (코웨이, 쿠쿠, SK매직) | 이사 예정 고객 사전 타겟팅 -> CAC 절감 | 높음 (현 방문판매 CAC 대비) |
| **통신사** (KT, SKT, LG U+) | 이사 고객 인터넷/IPTV 신규 가입 유치 | 높음 (통신 약정 LTV 높음) |
| **인테리어 플랫폼** (오늘의집, 집닥) | 이사 예정 고객에게 인테리어 서비스 추천 | 중간 (시공 중개 수수료 기반) |
| **건설사/분양사** | 이사 수요 예측, 분양 타겟 마케팅 | 높음 (직방RED 사례 입증) |
| **금융사** (은행, 보험) | 이사 대출, 화재보험, 이사보험 교차판매 | 중간~높음 |
| **가구/생활용품 기업** | 이사 시점 맞춤 프로모션 | 중간 |
| **지자체/공공기관** | 인구 유입 예측, 인프라 계획 | 중간 (공공 예산 기반) |

### 7.3 Snowflake 기반 차별화 포인트

1. **데이터 마켓플레이스**: Snowflake Marketplace를 통해 이사 인텔리전스 데이터를 B2B로 직접 유통 가능
2. **데이터 클린룸**: 통신사/렌탈사/부동산 플랫폼 간 개인정보 보호를 유지하면서 데이터 결합 분석 가능
3. **실시간 파이프라인**: 전입신고, 부동산 거래 등의 데이터를 실시간 파이프라인으로 처리
4. **예측 모델링**: Snowpark ML로 이사 확률 예측 모델 구축 및 B2B 고객에게 API로 제공

### 7.4 시장 기회 규모 추정

이사 트리거 연관 소비 시장 50조 원 이상에서, B2B 데이터/인텔리전스 서비스가 차지할 수 있는 비중을 보수적으로 0.1~0.5%로 추정하면:

| 시나리오 | TAM (총 시장) | 데이터 서비스 비중 | SAM |
|---|---|---|---|
| 보수적 | 50조 원 | 0.1% | **500억 원** |
| 기본 | 70조 원 | 0.3% | **2,100억 원** |
| 낙관적 | 100조 원 | 0.5% | **5,000억 원** |

> 직방RED가 12억 원에서 116억 원으로 1년 만에 10배 성장한 사례는, 한국 시장에서 부동산/이사 관련 B2B 데이터 서비스의 수요가 폭발적으로 존재함을 보여준다.
