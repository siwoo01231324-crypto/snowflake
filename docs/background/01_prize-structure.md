# 무빙 인텔리전스 — 수익 구조 & 시장 규모 분석

> 작성일: 2026-04-06
> 목적: 서비스의 수익 모델, 타겟 고객별 시장 규모(TAM/SAM/SOM), 단위 경제학, 가격 전략을 숫자 기반으로 정리

---

## Executive Summary

무빙 인텔리전스는 통신 주소변경 데이터를 활용해 이사 2~4주 전에 수요를 예측하고, 이를 B2B로 판매하는 데이터 플랫폼이다. 타겟 시장은 **가전렌탈(12조) + 인테리어(37조) + 이사서비스(3.5조) + 통신(마케팅비 8조) + 부동산 플랫폼 + 금융/보험**으로, 이사를 트리거로 한 연관 소비 시장은 **연 50조원 이상**이다. 이 시장의 B2B 데이터 지출 중 우리가 점유할 수 있는 SOM은 **Year 1: 13억, Year 3: 70~100억**으로 추정한다.

## 2. TAM / SAM / SOM 계산

### Bottom-up 방식 (기업 수 × ACV)

| 세그먼트 | 잠재 기업 수 | 예상 ACV | SAM |
|----------|------------|---------|-----|
| 가전렌탈 (대형 4사) | 4 | 1~3억 | 4~12억 |
| 가전렌탈 (중소) | 10 | 3,000~5,000만 | 3~5억 |
| 인테리어 플랫폼 (대형) | 4 | 5,000만~2억 | 2~8억 |
| 인테리어 업체 (중견) | 20 | 1,000~3,000만 | 2~6억 |
| 이사 플랫폼 | 5 | 2,000~5,000만 | 1~2.5억 |
| 통신사 | 4 | 1~5억 | 4~20억 |
| 부동산 플랫폼 | 5 | 2,000만~1억 | 1~5억 |
| 금융/보험 | 10 | 3,000만~1억 | 3~10억 |
| **합계** | **~62개** | — | **20~68.5억** |

### Top-down 방식

| 단계 | 계산 | 금액 |
|------|------|------|
| **TAM** | 이사 연관 시장 전체 B2B 마케팅 지출 | **약 1.5조원/년** |
| **SAM** | 디지털 CRM/마케팅 인프라 보유 + 데이터 구매 의사 있는 기업 | **약 200~500억원/년** |
| **SOM (Year 1)** | 현실적 점유 (37개 고객 × 평균 ACV 3,500만) | **약 13억원** |
| **SOM (Year 3)** | 200개 고객 × 평균 ACV 4,500만 | **약 90억원** |

> Bottom-up(20~68.5억)과 Top-down SAM(200~500억)이 합리적 범위 내에서 수렴. Year 1 SOM 13억은 SAM의 2.6~6.5% — B2B SaaS 초기 점유율 기준(1~5%) 내 현실적 수치.

|-----------|------------|---------|------|

## 4. 단위 경제학 (Unit Economics)

### 4-1. 벤치마크 기반 목표

| 지표 | 글로벌 벤치마크 (2025) | 우리 목표 |
|------|---------------------|---------|
| **CAC** (SMB) | $200~$300 | 300~500만원 |
| **CAC** (Mid-Market) | $1,400~$5,300 | 1,000~3,000만원 |
| **CAC** (Enterprise) | $2,200~$14,700 | 2,000~5,000만원 |
| **LTV:CAC** | 3.6:1 (중간값) | 목표 ≥ 3:1 |
| **CAC Payback** | 20개월 (중간값) | 목표 ≤ 15개월 |
| **NRR** | SMB 97% / Mid 108% / Ent 118% | 목표 ≥ 110% |
| **Gross Margin** | 77~80% (순수 SW) | 목표 ≥ 75% |
| **ARR/Employee** | $129K (중간값) | 목표 ≥ 1.5억원 |

### 4-2. 시나리오별 단위 경제학

**Starter 고객 (월 100만원)**
- ACV: 1,200만원
- 추정 CAC: 400만원 (콘텐츠 마케팅 + 인바운드)
- LTV (평균 계약 3년): 3,600만원
- LTV:CAC = **9:1** (건강)
- Payback = **4개월**

**Growth 고객 (월 300만원)**
- ACV: 3,600만원
- 추정 CAC: 1,500만원 (PoC + 직접 영업)
- LTV (평균 계약 2.5년): 9,000만원
- LTV:CAC = **6:1** (건강)
- Payback = **5개월**

**Enterprise 고객 (월 1,000만원+)**
- ACV: 1.2억원+
- 추정 CAC: 3,000만원 (PoC 3개월 무료 + 전담 영업)
- LTV (평균 계약 3년): 3.6억원+
- LTV:CAC = **12:1** (매우 건강)
- Payback = **3개월**

> PoC → 유료 전환율 벤치마크: B2B SaaS 20~30%. Enterprise $100K+ 딜 win rate: 15~25%.
