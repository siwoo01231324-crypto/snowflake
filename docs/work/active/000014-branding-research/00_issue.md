# chore: 무빙 인텔리전스 브랜딩 전략 리서치 — 네이밍·브랜드 정체성·후킹 포인트

## 목적
해커톤 수준의 프로젝트명('무빙 인텔리전스')을 넘어, 실무 B2B SaaS 제품으로서 통할 수 있는 브랜딩 전략을 수립한다.

## 배경
기획·사업화 플랜(docs/background/)이 완성된 시점에서, 본격적인 제품화를 앞두고 네이밍·브랜드 정체성·타겟별 후킹 포인트를 실무자 레벨에서 정의해야 한다. 국내외 데이터 B2B 브랜딩 사례를 리서치하고, 프로젝트에 최적화된 브랜딩 방향을 도출한다.

## 완료 기준
- [ ] 프로젝트 특성에 맞는 네이밍 후보 5개+ 도출 (영문/한글, 도메인 가용성 포함)
- [ ] B2B 타겟별 후킹 포인트 정리 (Tier 1 기준 — 렌탈사·인테리어·이사 플랫폼)
- [ ] 국내외 유사 데이터 B2B 브랜딩 벤치마크 (Porch Group Media, 직방RED 등 포함)
- [ ] 브랜드 정체성 방향성 정의 (포지셔닝, 톤앤매너)
- [ ] 결과물 docs/background/ 에 저장

## 구현 플랜
1. docs/background/ 기존 리서치 문서 검토 (사업 모델·경쟁 환경 파악)
2. 국내외 데이터 B2B 서비스 브랜딩 사례 리서치 (Porch Group Media 등)
3. 네이밍 후보 도출 — 핵심 가치 제안("이사 예측", "선행지표", "Pre-Mover") 기반
4. 타겟 세그먼트별 후킹 포인트 정의
5. 브랜드 정체성 방향성 문서화
