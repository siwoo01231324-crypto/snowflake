## Executive Summary

무빙 인텔리전스의 브랜딩 전략을 **실무자 수준**으로 체계적으로 수립한다. 10년차 브랜딩 팀장 관점에서, 글로벌 브랜딩 프레임워크·네이밍 방법론·경쟁사 벤치마크·유저/기업 저니 설계를 종합하여 **카테고리 크리에이터**로서의 브랜드 정체성을 확립한다.

## 실행 전략 — 3단계 접근

### Phase 1: 프레임워크 기반 전략 수립

**적용 프레임워크:**

| 프레임워크 | 적용 목적 | 핵심 산출물 |
|-----------|---------|-----------|
| **Category Design** (Play Bigger) | 한국 최초 Pre-Mover 데이터 카테고리 정의 | 카테고리 POV, 카테고리명 |
| **Keller CBBE Pyramid** (B2B 변형) | 브랜드 자산 구축 로드맵 | 4단계 브랜드 빌딩 플랜 |
| **Aaker Brand Identity Model** | 브랜드 정체성 구성요소 정의 | 브랜드 에센스, 핵심 가치 |
| **LinkedIn 95-5 Rule** | B2B 장기 브랜드 빌딩 전략 | 60/40 예산 배분, 메모리 구축 |
| **Hook Point** (Brendan Kane) | 3초 안에 주의를 끄는 메시지 | 세그먼트별 후킹 메시지 |

**핵심 전략 방향:**
1. **카테고리 크리에이션**: "이사 예측 데이터" 카테고리 자체를 창조 → 카테고리 킹 선점
2. **95-5 Rule 적용**: B2B 잠재 고객의 95%는 지금 당장 구매하지 않음 → 장기 브랜드 빌딩으로 Mental Availability 확보
3. **신뢰 기반 포지셔닝**: B2B에서는 감정(Feelings)보다 신뢰성(Credibility)과 성과(Performance)가 핵심

### Phase 2: 경쟁사 벤치마크 & 포지셔닝

**글로벌 벤치마크 대상:**

| 기업 | 브랜딩 전략 핵심 | 시사점 |
|------|---------------|--------|
| **Porch Group Media** | "Where people live matters" — 이사 데이터 = 생활 인사이트 | 가장 직접적 벤치마크. 데이터→인사이트→행동 프레이밍 |
| **Snowflake** | "The Data Cloud" — 은유형 네이밍 + 카테고리 재정의 | 추상적 이름이 카테고리를 정의한 성공 사례 |
| **ZoomInfo** | "Revenue Operating System" — 기능형→플랫폼 전환 | 데이터 벤더→플랫폼으로 진화한 리포지셔닝 |
| **6sense** | "Revenue AI" — AI+예측 포지셔닝 | 예측 데이터의 AI 프레이밍 참고 |
| **직방RED** | B2C→B2B 확장, 부동산 데이터 수익화 | 국내 B2B 데이터 브랜딩 선례 |

**포지셔닝 전략:**
- 2축 포지셔닝 맵: **예측 선행성(Pre-Mover vs Post-Mover)** × **데이터 활용 범위(광고 한정 vs CRM 전채널)**
- 무빙 인텔리전스의 고유 포지션: **"유일한 Pre-Mover + CRM 직접 주입"** = 빈 사분면 점유

### Phase 3: 유저/기업 End-to-End 저니 & 후킹 포인트

**B2B 고객 저니 8단계:**
```
Awareness → Interest → Evaluation → Trial(PoC) → Purchase → Onboarding → Expansion → Advocacy
```

**세그먼트별 후킹 포인트 설계:**
- 각 Tier 1 세그먼트(렌탈사, 인테리어, 이사 O2O, 통신사)에 최적화된 3초 후킹 메시지
- Pain → Gain 전환 프레임워크
- 엘리베이터 피치 (30초/60초/3분)

**브랜드 터치포인트 전체 설계:**
- 디지털(웹사이트, 대시보드, API 문서, 이메일)
- 오프라인(해커톤, 영업 미팅, 컨퍼런스)
- 콘텐츠(블로그, 인사이트 리포트, 케이스 스터디)
