# src/app — Streamlit 앱 루트

## 목적

Snowflake 기반 이사 수요 분석 Streamlit 앱의 루트 디렉토리.
3탭 구조. Streamlit in Snowflake(SiS) + 로컬 개발(`local_run.py`) 양쪽 실행 지원.

## 로컬 Streamlit 실행 방법

> Python 3.14 환경. `import streamlit`이 WMI COM 쿼리로 무한 대기하는 버그 있음 → 아래 방법으로 우회.

### 1. 래퍼 스크립트 수정 (포트 변경 시만)

`C:\Users\watch\AppData\Local\Temp\sl_wrapper.py` 에서 포트 번호 수정:
```python
sys.argv = ['streamlit', 'run', 'local_run.py', '--server.port', '8504', ...]
```

### 2. 실행

Claude Code 프롬프트에서:
```
! python C:\Users\watch\AppData\Local\Temp\launch_final2.py
```

출력된 PID 확인 후 브라우저에서 `http://localhost:8504` 접속.

### 3. 로그 확인

```
! tail -20 C:\Users\watch\AppData\Local\Temp\sl_final.log
```

### 4. 포트 사용 중 확인

```
! netstat -ano | findstr :8504
```

### 5. 프로세스 종료

```
! kill -9 <PID>
```

### 핵심 파일 2개

| 파일 | 역할 |
|------|------|
| `C:\Users\watch\AppData\Local\Temp\sl_wrapper.py` | `platform._wmi=None` 우회 + `sys.stdout` 로그 리다이렉트 + streamlit 실행 |
| `C:\Users\watch\AppData\Local\Temp\launch_final2.py` | `CREATE_NEW_PROCESS_GROUP`으로 백그라운드 subprocess 실행 |

> **왜 이 방법?** Claude Code bash는 TTY 없음 + Python 3.14 `_wmi` C 확장이 WMI COM 쿼리 무한 대기
> → `platform._wmi = None`을 `import streamlit` **이전**에 설정해야 함
> → Windows 프로세스는 cygwin fd를 상속 못 하므로 Python 내부에서 직접 파일로 stdout 리다이렉트

## tabs/heatmap.py 기능

- **choropleth 지도**: pydeck `GeoJsonLayer` + `V_SPH_REGION_MASTER.DISTRICT_GEOM → ST_ASGEOJSON()`
- **시그널 필터**: CONTRACT_COUNT (선행) / OPEN_COUNT (후행)
- **연월 필터**: `V_TELECOM_NEW_INSTALL` 기준 최근 24개월 selectbox
- **Dual-Tier 배지 범례**:
  - MULTI_SOURCE (중구·영등포구·서초구, CITY_CODE 11140·11560·11650): 4종 시그널
  - TELECOM_ONLY (22구): 통신 단일 시그널
- **구별 수치 테이블**: expander 내 정렬 테이블
- **예측값 조회**: `PREDICT_MOVE_DEMAND(city_code, YYYYMM)` UDF 버튼 실행 (#23)

## cortex.py 공개 API

| 함수 | 용도 | 소비 이슈 |
|------|------|-----------|
| `get_district_insight(session, city_code, year_month)` | V_AI_DISTRICT_INSIGHTS 단일 행 조회 | #28 히트맵 상세 패널 |
| `classify_demand(session, city_code, year_month)` | V_AI_DEMAND_GRADE 단일 행 조회 | #28 색상/범례 |
| `find_similar_districts(session, city_code, top_k=5)` | SP_FIND_SIMILAR_DISTRICTS 호출 | #29 유사 지역 탭 |
| `aggregate_state_summary(session, year_month)` | V_AI_STATE_SUMMARY 단일 월 조회 | #29 보조 카드 |
